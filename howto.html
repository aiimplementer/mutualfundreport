Generate Your Portfolio Report in 4 Simple Steps
Follow these simple steps to generate a detailed analysis of your mutual fund portfolio.
1
Download Your CAS Statement
Download your Consolidated Account Statement (CAS) from CAMS or KFintech. If you don’t already have one, you can generate it from the CAMS investor portal here: Generate CAS
2
Save a Password-Free Copy
CAS files are usually password protected. Open the file and use Print → Save as PDF to create a new copy without a password so it can be processed.
3
Upload the CAS File
Go to the Dashboard and upload the CAS PDF file that you saved.
4
Generate Your Portfolio Report
Choose the report you want such as My Portfolio View or Portfolio Overview. Your portfolio will be analyzed and the report will be generated within seconds.
💡 Tip: You can generate multiple reports using the same uploaded CAS file to analyze your portfolio from different perspectives.
ℹ️ Note: Please ensure the CAS PDF is not password protected, as protected files cannot be processed.
